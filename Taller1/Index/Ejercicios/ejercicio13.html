Ejercicio de Uso de Iframes

Google Maps

Contenido Externo

El uso de <iframe> en HTML5 permite incrustar contenido externo dentro de una página web. Esto es útil para mostrar mapas, videos, páginas web de terceros y más, sin necesidad de que el usuario abandone el sitio actual. Se puede configurar el tamaño, permitir pantalla completa y especificar diversas propiedades para mejorar la visualización.
